# Instrucciones para agentes

Antes de modificar cualquier parte del proyecto, cualquier agente debe leer completo el archivo `docs/skills/SPJ_REFACTOR_SKILL.md` y seguir sus reglas obligatorias.

Este repositorio está en una fase de refactor controlado: no se debe modificar lógica de negocio, refactorizar módulos ni eliminar código legacy sin cumplir el flujo, protecciones y checklists documentados en ese skill.
