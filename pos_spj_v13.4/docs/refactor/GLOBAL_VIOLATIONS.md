## Formato de hallazgo

Cada hallazgo debe registrar:

```text
ID:
Severidad:
Categoría:
Módulo:
Archivo:
Línea:
Descripción:
Causa raíz:
Test de protección:
Estado:
Iteración detectada:
Iteración corregida:
```

## Regla

No borrar hallazgos corregidos.

Cambiar su estado a `RESOLVED` para conservar trazabilidad.
